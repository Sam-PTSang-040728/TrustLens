
1. Giảng viên đăng nhập vào hệ thống TrustLens.
2. Giảng viên tạo hoặc chọn lớp/học phần và bài nộp cần kiểm tra.
3. Giảng viên upload một hoặc nhiều file báo cáo PDF/DOCX của sinh viên.
4. Hệ thống kiểm tra file: định dạng, dung lượng, khả năng đọc văn bản, trùng lặp file.
5. Hệ thống trích xuất text và phát hiện phần tài liệu tham khảo.
6. Hệ thống tách từng citation, chuẩn hóa trường thông tin và phát hiện phong cách citation.
7. Hệ thống xác minh DOI/tiêu đề/tác giả/năm qua metadata API hoặc nguồn mở hợp pháp.
8. Hệ thống đánh giá độ tin cậy, độ cập nhật, độ phù hợp và định dạng citation.
9. Hệ thống tính Trust Score, gắn nhãn cảnh báo và sinh kết luận.
10. Giảng viên xem dashboard, mở chi tiết từng citation, điều chỉnh hoặc ghi chú nếu cần.
11. Giảng viên xuất báo cáo kết quả để lưu, phản hồi hoặc đưa vào minh chứng chấm điểm.
